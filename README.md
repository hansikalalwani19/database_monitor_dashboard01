# Database Monitor Dashboard

A modern database monitoring dashboard built using React, TypeScript, Vite, Tailwind CSS, and Recharts. It provides an interactive interface to monitor database health, performance, alerts, and analytics.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts

## Features

- Dashboard Overview
- Database Inventory
- Health Monitoring
- Alerts & Incidents
- CPU, Memory & Connection Analytics
- Responsive UI

## Setup

```bash
git clone https://github.com/hansikalalwani19/database_monitor_dashboard01.git
cd database_monitor_dashboard01
npm install
npm run dev
```
